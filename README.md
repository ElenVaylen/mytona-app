Тестовое задание mytona

React(create-react-app) с использованием redux, react router, redux saga

Запуск:
npm install
npm start

Или: https://elenvaylen.github.io/mytona-app/build

На главной просто текст для заполнения пустоты. 
Страница новостей использует апи с ресурса https://newsapi.org/
Регистрация использует localstorage как хранилище,
а при логине проверяется есть ли там зарегистрированный email.
